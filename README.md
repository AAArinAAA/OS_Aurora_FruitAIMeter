# Разработано в Омском Государственном Техническом Университете

## Структура проекта:

- icons/  - Содержит иконки для проекта ращличных размеров
- images/ - Содержит изображение яблока 
- qml/ - Соедржит исходный QML код элементов и UI компоненты
- rpm/ - Содержит настройки сборки rpm-пакета
- src/ - Содержит C++ исходный код 
	- main.cpp - Файл, являющийся точкой входа в приложение
